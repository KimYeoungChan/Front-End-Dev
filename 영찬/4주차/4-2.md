# 틱택톡 게임
- 삼목(3x3 표 위에서 진행하는 게임)
- HTML을 JavaScript로 작성하기
- 이차원 배열을 for문으로 작성
- 구조분해 할당(distructuring)

```
객체의 구조분해 할당
const body = document.body;
const { body } = document;
```

```
배열의 구조분해 할당
const [one, two, three, four, five] = arr;
const [one, three, five] = arr; // 2,4 사용하지 않고 싶을 때
```

```
const obj = {a: 1, b: 2};

const { a, b } = obj; // 다음 두 줄을 이렇게 한 줄로 표현 가능
const a = obj.a;
const b = obj.b;
```
